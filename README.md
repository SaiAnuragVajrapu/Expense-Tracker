# Expense Tracker Web App

A simple and responsive Expense Tracker Web Application built using HTML, CSS, and JavaScript. This application helps users manage their income and expenses efficiently with CRUD operations and export functionality.

---

# Features

## Transaction Management
- Add new transactions
- Edit existing transactions
- Delete transactions
- View transaction history

## Financial Summary
- Real-time balance updates
- Separate income tracking
- Separate expense tracking

## Export Functionality
- Export transaction data to PDF
- Export transaction data to Excel

## Data Persistence
- Uses Local Storage to save transactions
- Data remains available after refreshing the browser

---

# Technologies Used

- HTML5
- CSS3
- JavaScript
- jsPDF Library
- SheetJS (XLSX)
